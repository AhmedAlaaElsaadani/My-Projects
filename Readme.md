Full-stack developer focused on **modern, multilingual web products**. I build polished front-ends with **React** and **Next.js**, backed by **ASP.NET** — shipping company websites, e-commerce, SaaS platforms, and real-time AI features for clients across the **MENA region**.

- 🚢 **Delivery** — from first commit to production deployment
